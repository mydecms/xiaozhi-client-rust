# xiaozhi-client-rust
虾哥的小智AI桌面客户端


使用 rust 构建的 TUI 界面的 小智 AI 客户端。
=== 

开发语言：Rust

运行方式：打开 Windows Terminal 客户端运行 exe 文件。

执行文件：xiaozhi-tui-cursive.exe

主页： https://github.com/mydecms/xiaozhi-client-rust


当前功能列表：
=== 

1. 输入文字，收听小智回应；

2. 输入语音对话， 点击快捷键 r 开始录音，再次点击 r 停止录音；

3. 服务器回传的文字、 Emojo 状态不显示；
